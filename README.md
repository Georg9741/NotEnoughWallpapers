# NotEnoughWallpapers
A Rainmeter Skin that let's you set predefined Wallpaper Engine Profiles and Rainmeter Layouts

&nbsp;

NotEnoughWallpapers Setup:
1. Write the Execution Path of your WallpaperEngine in WEPath, in Variables.inc (if it's not the Default Path)
2. Create Profiles in Wallpaper Engine to run them with the Menu,  
   name them *profileNUMBER* or edit Variables.inc (Section: Rename Stuff)
3. Create Rainmeter Layouts in the Rainmeter Menu in the "Themes" Section,  
   name them *profileNUMBER* or edit Variables.inc (Section: Rename Stuff)
4. Configure the Design of NEW (NotEnoughWallpapers) in Settings.ini to your liking  
   (accessible from the Context Menu)
5. Change the Preview Image for each Profile via the Context Menu
